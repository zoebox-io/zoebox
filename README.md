Postgres, MySQL, Redis, MongoDB, Qdrant, Redpanda, and MinIO in your menubar. Click to start, click to connect.

## Install

[Download the latest release](https://zoebox.io/#download) — requires macOS 14+ with Apple Silicon.

## Services

| Service | Default Port | Versions |
|---------|-------------|----------|
| PostgreSQL | 5432 | 14, 15, 16, 17, 18 |
| MySQL | 3306 | 8.4, 9 |
| Redis | 6379 | 6, 7, 8 |
| MongoDB | 27017 | 7.0, 8.0 |
| Redpanda | 9092 | 24, 25 |
| Qdrant | 6333 | 1.15, 1.16, 1.17 |
| MinIO | 9000 | 2024.10, 2025.04, 2025.09 |

## How it works

1. **Install** — drag to Applications
2. **Add a service** — pick from the menubar
3. **Connect** — copy the connection string into your app

No Docker. No config files. No daemons.
